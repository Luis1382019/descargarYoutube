# Como usar?
**Instalar venv**  
    - `sudo apt install python3-venv`
  
**Crear entorno virtual**  
    - `python3 -m venv myenv`

**Activar entorno**  
    - `source myenv/bin/activate`

**Instalar librerias**  
    - `pip install yt-dlp`

**Ejecutar archivo**  
    - `python3 downloadYT.py`

**Desactivar entorno**  
    - `deactivate`
